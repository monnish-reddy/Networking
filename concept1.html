üß†Essential Networking Concepts

üåçPublic IP

Globally unique, routable IP address assigned by ISP or cloud provider for internet access.

üè†Private IP

Internal network IPs (RFC 1918) not routable on internet: 10.x.x.x, 172.16.x.x, 192.168.x.x

üîóCIDR Notation

Defines subnet size using slash notation (e.g., /24 = 255.255.255.0 subnet mask)

‚ö°DHCP

Automatically assigns IP addresses, subnet masks, and gateway info to network devices

üîÑNAT/PAT

Translates private IPs to public IP, with PAT adding unique ports for each device

The NAT/PAT Process:

  1. Laptop requests https://google.com
  2. Router translates private IP (192.168.1.10) to public IP (105.23.17.8)
  3. Router assigns unique port (50432) using PAT
  4. Google responds to 105.23.17.8:50432
  5. Router forwards response back to 192.168.1.10

‚ú®Magic of PAT

Multiple devices share the same public IP simultaneously! Each gets a unique port number, allowing the router to track and route responses correctly.
